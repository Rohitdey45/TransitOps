Login

Authenticate and receive a JWT. The token is stored automatically and reused on every request below.
POST /auth/login Public

Current User

Verifies the stored token and returns the decoded payload.
GET /auth/me Authenticated

List Vehicles

GET /vehicles All roles

Register Vehicle

POST /vehicles Fleet Manager

Update Vehicle

PATCH /vehicles/:id Fleet Manager

Delete Vehicle

DELETE /vehicles/:id Fleet Manager

Vehicle Operational Cost

GET /vehicles/:id/operational-cost All roles

List Drivers

GET /drivers All roles

Register Driver

POST /drivers Fleet Manager

Update Driver

PATCH /drivers/:id Fleet Manager (full) / Safety Officer (status + safetyScore only)
Log in as Safety Officer and try changing Name too — it should be silently ignored, only status/safetyScore applied.

Delete Driver

DELETE /drivers/:id Fleet Manager

List Trips

GET /trips All roles

Create Trip (Draft)

POST /trips Fleet Manager & Driver

Dispatch Trip

Flips vehicle/driver status to ON_TRIP and records starting conditions.
PATCH /trips/:id/dispatch Fleet Manager & Driver

Reassign Trip

Change driver/vehicle on a Draft or Dispatched trip. If Dispatched, the old pair is released and the new pair is booked automatically.
PATCH /trips/:id/reassign Fleet Manager & Driver

Complete Trip

PATCH /trips/:id/complete Fleet Manager & Driver

Cancel Trip

PATCH /trips/:id/cancel Fleet Manager & Driver

List Maintenance Records

GET /maintenance All roles

Create Maintenance Record

Flips vehicle status to IN_SHOP automatically.
POST /maintenance Fleet Manager & Safety Officer

Close Maintenance Record

Restores vehicle status to AVAILABLE (unless Retired).
PATCH /maintenance/:id/close Fleet Manager & Safety Officer
